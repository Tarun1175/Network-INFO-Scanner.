# Network Information Scanner

A beginner Python networking and cybersecurity project.

## Features
- Displays hostname
- Shows local IP address
- Shows operating system information
- Displays public IP address
- Simple terminal-based scanner

## Technologies Used
- Python
- socket
- platform
- requests

## Sample Output

Hostname: localhost
Local IP Address: 127.0.0.1
System: Linux
Release: 4.19.136-perf+
Public IP Address: 223.xxx.xxx.xxx

## Purpose
This project was created to practice Python scripting, networking basics, and cybersecurity-related concepts.
